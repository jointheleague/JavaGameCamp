Tash Me

Steps:

  1. Choose a picture from "img" folder inside of _2_tash_me
  2. Open Tash_Me.pde using Processing.
  3. Drag the image you chose in Step 1 into your sketch
  4. On line 1 of your sketch, add the following code: 
    PImage mustache;
PImage friend;
  5. Load your image by adding the following code to the setup() method:
    friend = loadImage("friend.jpg");
size(800, 600);
friend.resize(width,height);
  6. Change “friend.jpg” in the above code so that it matches the name of your image.
  7. At the beginning of the draw() method, set “friend” as your background using the background() method.
  8. Run the program. Is your image displayed in the sketch? If not, get this part working before you move on to the next step.

  9. Find "mustache.png" in the from "img" folder inside of _2_tash_me, and drag and drop it into your processing window.
  10. In the setup() method, load the mustache image using the following code: 
    11.  mustache = loadImage("mustache.png");
  11. Change “mustache.png” from the previous step so that it matches the name of your image and resize it if you need to (see step 5 for a hint on how to resize an image).
  12. In the draw() method, add the mustache to your sketch using: 
    13. image(mustache, 150, 400);
  13. Run the program. Is your mustache displayed on the face? If not, get this part working before you move on to the next step.
  14. Change the previous line of code so that the mustache moves with your mouse! Hint: Processing gives you mouseX and mouseY variables to help you do this.
  15. Use an if statement to make the mustache appear only when the mouse is pressed. Hint: Processing gives you a mousePressed variable to help you do this.
  16. Inside of your if statement, redraw the background before you draw the mustache. This will make it so that only one mustache appears at a time.
  17. EXTRAS

    • Add or subtract a number from mouseX and mouseY when drawing the mustache so the middle appears exactly where you click the mouse.
    • Draw a hat on your image when the right mouse button is clicked using: 
      if (mouseButton == RIGHT){
