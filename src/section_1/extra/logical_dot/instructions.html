Logical Dot

Steps:

  1. Find the starter code for this recipe and open it using Processing (Hint: the file name is Logical_Dot.pde) and it should be in the section_1 folder.
  2. In the setup() method, set the size of your sketch to 600 by 600 
    size(600,600);
  3. Draw an ellipse inside the draw() method. 
    ellipse(300, 300, 100, 100);
}
  4. Make your ellipse a nice color. 
    fill(52, 213, 235);
  5. In the draw() method, make it so that if the mouse is pressed the color of the ellipse changes. 
    if (mousePressed) {
        // fill with one color here
} else {
        // fill with a different color here
}
  6. Create a variable on line 1 (outside of the setup() and draw() methods) that will old the x-position of your dot 
    int x = 300
}
  7. Update the ellipse() method to use to x variable you created as the x-position for the ellipse
  8. in the if-statement you created above, increase the value of x when the mouse is pressed 
    x = x + 1;
  9. If the dot moves completely off of the screen, make the dot reenter on the left side of the screen 
    if(x>=600+50){
    x = 0 - 50;
}
