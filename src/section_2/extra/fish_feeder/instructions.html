Fish Feeder

Steps:

  1. Open FishFeeder.pde using Processing
  2. In the setup() method, set your canvas size. 
    size(600,600);
  3. At the top of your program (on line 1) create variables for the location of the fish food. 
    int foodX = 100;
int foodY = 20;
  4. In the draw() method, make the background the color of water 
    background(141,233,237);
  5. Draw an ellipse to represent a piece of fish food, using the variables from step 3 as the x and y positions. 
    6. fill(212, 152, 42);
noStroke();
ellipse(foodX, foodY, 30, 30);
  6. In the draw() method, update the value of foodY to make it fall down the screen 
    foodY = foodY + 1;
  7. Using an if-statement and the build-in height variable, reset the value of foodY when it reaches the bottom of the screen
  8. When the food leaves the screen, use the code below to set foodX to a random value. 
    foodX = (int) random(width-20);
  9. Add this drawFish() method at the very bottom of your sketch: 
     void drawFish(){
      noStroke();
      fill(255,255,255);
      ellipse(mouseX-17,mouseY-17,20,20); //back side eye
      fill(255,200,88);
      ellipse(mouseX,mouseY,90,50); //body
      triangle(mouseX+30,mouseY,mouseX+70,mouseY+30,mouseX+70,mouseY-30); //tail
      stroke(0);
      triangle(mouseX,mouseY,mouseX+15,mouseY+10,mouseX+15,mouseY-10); //side fin
      noStroke();
      fill(255,200,88);
      ellipse(mouseX,mouseY,15,15); //side fin cover
      noStroke();
      fill(255,255,255);
      ellipse(mouseX-25,mouseY-15,20,20); //front eye
      fill(0,0,0);
      ellipse(mouseX-25,mouseY-15,5,5); //pupil
      fill(141,233,237);
      ellipse(mouseX-45,mouseY,25,25); //mouth
}
  10. After you call background() inside of your draw() method, use this code to draw the fish: 
    drawFish();
  11. Above the setup() method, add an int variable to hold the score. 
    int score = 0;
  12. Add this checkCatch() method to the bottom of your program. 
    void checkCatch(){
      if(foodX > mouseX && foodX < mouseX+70){
           if(foodY > mouseY-17 && foodY <mouseY+30){
              score++;
              foodY = 20; //if the food was eaten, make more food fall from the top
              foodX = (int) random(width-20);
           }
      }
      else if (foodY > height && score > 0){
           score--;
      }
}
  13. Call the checkCatch() method inside your draw() method. 
    checkCatch()
  14. Show the score on the screen. 
    fill(0, 0, 0);
textSize(16);
text("Score: " + score, 20, 20);
  15. Decide how many points the user needs to score to win. Use an if-statement to check if they have reached that score, then stop drawing food and put up a message to tell the user they have won. 
    textSize(16); //if it’s not big enough
text("You Won!", 100, 100);
  16. COOL VARIATIONS
    • Make the speed of the fish food increase as the score increases.
    • Make some of the food poisonous (negative points if you catch it).
    • Make the fish get fatter as he eats more food.
    • Add text at the beginning to explain the game.
    • Add messages to encourage the user to finish!
    • Launch your fish using the following steps!
      1. Create a fishX variable at the top of your sketch with your other variable.
      2. Initialize the value of fishX to "width-100". This will be used to keep track of the x location of the fish.
      3. Replace all of the mouseX’s in your program with your new fishX variable.
      4. Your fish should now be stuck on the right side of your screen, but still able to move up and down.
      5. Create a boolean variable at the top of our sketch with the other variables. A boolean variable has only two possible values: true or false. 
        boolean launched = false;
      6. In your draw() method, if the mouse is pressed, change the value of launched to "true".
      7. Create another if-statement in your draw() method to check if launch is true. If it is, then increase the value of fishX.
      8. If the fishX is less than 0, set launched to "false" and fishX back to its original value.
