Googly Eyes

Steps:

  1. Open GooglyEyes.pde using Processing.
  2. Choose a face image from the "img" folder in _2_googly_eyes, then drag and drop it into your processing window
  3. On line 1 of your sketch (outside of the methods) declare a variable for the image: 
    PImage face;
  4. In your setup() method, import your image using the following code: 
    face = loadImage(“face.jpg”);
  5. Change the code you just entered so that “face.jpg” matches the name of the picture you chose.
  6. Set the size of your sketch and the size of your image to be the same by entering the following code in the setup method. 
    size(800,600);
face.resize(width,height);
  7. In the draw() method, place a white ellipse over the left eye of your image.
    HINT: To find out where to put it, add code to print the mouseX and mouseY where you click the mouse. 
    if(mousePressed){
    println("Mouse’s x-position: " + mouseX + "\n" + "Mouse’s y-position: " + mouseY + "\n");
}
  8. Now add a smaller black ellipse to the left eye to represent the pupil
  9. Add a white ellipse over the right eye using the same technique used for the left eye.
  10. Use mouseX and mouseY to make left pupil follow the mouse.
  11. In your head, find the difference (using subtraction) between the left eye's x-position and right eye's x-position, and remember the value.
  12. In your head, find the difference (using subtraction) between the left eye's y-position and right eye's y-position, and remember the value.
  13. Draw a second black ellipse for the right pupil, making its position relative to the left pupil. Do this by setting the x and y values for the ellipse using mouseX + [x-position difference] and mouseY + [y-position difference].
  14. Here’s the tricky part: we now need to stop the pupils from traveling outside of the eyes. Luckily, since we are drawing the right pupil relative to the left pupil, we will only need to do the following steps for one eye. Use the image below as a reference for the following steps:
  15. First, imagine a rectangle inside the left eye, within which you want the pupil to stay.
  16. Taking advantage of the code that we added above to print out the location where the user clicks their mouse, click on the corners of your imaginary square and record the values. Notice that the values in the example image are nice and round, but yours will probably be less friendly.
  17. Write an if-statement before you draw your pupil that will keep the pupil from moving past the left edge of the eye, as the example image demonstrates. To do this, you will first find the smallest value for x that exists within your imaginary square. You will check if the mouse's x value is smaller than that, and if it is, set it back to the smallest value within your imaginary square.
  18. Add three more if-statements, one for each of the remaining directions that the pupil could leave the eye:
    • If mouseX is greater than the largest x-value within your imaginary square, then reset mouseX to the largest x-value within your square
    • If mouseY is less than the smallest y-value within your imaginary square, then reset mouseY to the smallest y-value within your square
    • Ff mouseY is greater than the largest y-value within your imaginary square, then reset mouseY to the largest y-value within your square
  19. Make sure you SAVE YOUR CODE when you are done.
  20. Extra: Make the color of the eyes change when the user moves the mouse, by using the mouseX and mouseY variables in the fill() for the eyes.
