Pizza

Steps:

  1. Open Pizza.pde using Processing.
  2. Set your canvas size in the setup method. 
    size(width, height);
  3. Draw a biscuit-colored ellipse for the pizza base Do this in the setup method. You can use R,G,B values or the Processing Color Selector Tool to help you find the right colors. 
    fill(color);
ellipse(x, y, width, height);
  4. Using the same x and y positions, draw a slightly smaller red ellipse for sauce.
  5. Using the same x and y positions, raw an even smaller yellow ellipse for the cheese.
  6. Choose a topping image from the "img" folder in _1_pizza, then drag and drop it into your processing window
  7. Make a PImage variable for each type of topping at the top of the sketch: 
    PImage pepperoni;
  8. Load the topping image (in the setup method): 
    pepperoni = loadImage("pepperoni.png");
  9. In the draw method, use the image command to draw the topping somewhere on your pizza: 
    image(pepperoni,200,200);
  10. In the setup method, resize your topping image to make it fit properly. 
    pepperoni.resize(30, 30);
  11. Repeat the image command to add the topping to your pizza multiple times.
  12. Repeat steps seven through ten to add another topping to your sketch
  13. Use mousePressed, mouseX, and mouseY to add the new topping to your pizza when and where the user clicks the mouse. 
    if(mousePressed){
    image(pepperoni,mouseX,mouseY);
}
  14. Make sure you SAVE YOUR CODE when you are done.
  15. EXTRA
    • Add a different topping with the right mouse click.

    • Use pizzaBox.png in the "img" folder to put your pizza in a box.
