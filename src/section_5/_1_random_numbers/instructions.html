Random Numbers

Steps:

  1. Inside the main method, create a Random, similar to how you would create a Robot. 
    Random randy = new Random();
  2. Use this Random to create a random number. 
    int number = randy.nextInt();
  3. Print out the random number. 
    System.out.println("Random number is " + number);
  4. Get another number from 0 to 100, and print it out. 
    int numberLessThan100 = randy.nextInt(101);
System.out.println("Random number from 0 to 100 is " + numberLessThan100);
  5. Get a third number between 100 and 110, and print it out. 
    int low = 100;
int high = 110;
int randomBetweenLowAndHigh = randy.nextInt((high-low)+1)+low;
System.out.println("Random number from 100 to 110 is " + randomBetweenLowAndHigh);
